Finger Excercise #0: Movie Reviews (5 puntos)


El objetivo de este ejercicio es familiarizarse con el set de datos que vamos a usar para el 
Trabajo Práctico de este cuatrimestre.


El set de datos se puede bajar de la página de kaggle: 
http://www.kaggle.com/c/word2vec-nlp-tutorial


Vamos a usar el larchivo labeledTrainData.tsv. Este archivo tiene un review por línea y cada línea
 tiene 3 capos separados por tabs: id, sentiment y review.
id es el id de la película, desconocemos cuál es la película.
sentiment: es el sentimiento del review +1 = positivo, -1 = negativo.


Tenemos en total 25000 reviews en este set de datos. En este primer ejercicio vamos a hacer 
un análisis exploratorio de los datos.


Pedimos que publiquen en el grupo de facebook de la materia los siguientes resultados:


* Cantidad de reviews positivos y cantidad de reviews negativos.
* Longitud promedio, máxima y mínima de los reviews (cantidad de caracteres)
* ¿Es verdad que todos los reviews que contienen la palabra “amazing” son positivos?


El alumno que resuelva primero correctamente este ejercicio obtiene 5 puntos.


Se puede usar cualquier lenguaje para resolver este ejercicio.
